# SuperApp Business Hub

A comprehensive business management superapp with 12 specialized department modules.

## Overview

SuperApp Business Hub provides an integrated platform for managing key business functions including Sales, Marketing, Strategy, Leadership, HR, Finance, and more — all in one unified interface.

## Tech Stack

- **Next.js 15** with App Router
- **React 19**
- **Tailwind CSS 4**
- **Static Export** on Vercel

## Features

- 12 department modules with dedicated interfaces
- Modern responsive dashboard design
- Business intelligence and analytics views
- Team collaboration tools

## Getting Started

```bash
npm install
npm run dev
```

## Deployment

Deployed on Vercel with static export.
